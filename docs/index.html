[ faq ]

Is it safe to give the bot my address?

Yes — we only read public blockchain data via TronGrid. The bot never asks for keys, seeds or signatures. Source code is public on GitHub.

How fast are the alerts?

Polling runs every 30 seconds. In practice you'll see a notification within 30–60s of the on-chain confirmation.

Which networks are supported?

Right now: USDT on Tron (TRC20). BSC, Ethereum and BTC are on the roadmap.

Can I self-host it?

Yes — the project is MIT and ships with docker-compose. See the README on GitHub.
